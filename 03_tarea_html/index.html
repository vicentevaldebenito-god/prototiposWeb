Introducción al lenguaje

HTML o HyperText Markup

Language es el lenguaje estándar utilizado para crear y estructurar el contenido de las páginas web.

Etiquetas de formato

La etiqueta <hr> pone una línea horizontal para separar secciones o temas distintos en la página. La etiqueta <br> solo hace un salto de línea dentro del mismo párrafo, sin crear uno nuevo.
